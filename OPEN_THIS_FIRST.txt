STEP 1: CHECK YOUR COMPUTER FIRST
═══════════════════════════════════════════════════════════════════════════════

Before we do anything else: how did you get hacked?

If you're not sure, there might still be malware on your computer. Moving your 
NFTs to a new wallet on an infected machine just gives the hacker access to 
that wallet too.

   ⚠  Run a malware scan before you proceed.
   
      Free option: https://www.malwarebytes.com
      
      Better option: Use a completely different device if you have one.

Once you're confident your machine is clean, continue to Step 2.


═══════════════════════════════════════════════════════════════════════════════
STEP 2: INSTALL NODE.JS (if you don't have it)
═══════════════════════════════════════════════════════════════════════════════

Node.js is what makes this tool run. If you've never heard of it, you probably
don't have it.

   1. Go to: https://nodejs.org
   
   2. Click the big green button that says "LTS" (the recommended version)
   
   3. Install it like any normal program — just click Next, Next, Finish
   
   4. You don't need to open it or do anything else. Just having it installed
      is enough.

How to know it worked:
   - The installer finished without errors
   - That's it. You won't see an app icon or anything.

STEP 3: GET AN ALCHEMY API KEY (free, takes 2 minutes)
═══════════════════════════════════════════════════════════════════════════════

Alchemy is the service that lets us scan the blockchain for your NFTs.
It's free. No credit card. Here's exactly what to do:

   1. Go to: https://dashboard.alchemy.com
   
   2. Click "Sign Up" — you can use Google or make a new account
   
   3. Once you're logged in, you'll see a dashboard. 
      Click "Create new app" (or "Apps" then "Create App")
   
   4. It'll ask for a name — type anything, like "rescue" or "lifeboat"
      The name doesn't matter.
   
   5. Click "Create"
   
   6. Now you'll see your app. Look for "API Key" in the top right area.
      Click the copy button next to it.
   
   7. Paste that key somewhere safe — you'll need it in a minute.

STEP 4: RUN LIFEBOAT
═══════════════════════════════════════════════════════════════════════════════

Now we're ready to actually run the tool.

   If you're on WINDOWS:
   
      1. Find the file called "start.bat" in this folder
      2. Double-click it
      3. A black command window will open — that's normal, leave it open
      4. Your web browser will open automatically to the tool
   
   
   If you're on MAC:
   
      1. Open the "Terminal" app (search for it in Spotlight)
      2. Type: cd 
         (that's "cd" then a space)
      3. Drag the LIFEBOAT folder into the Terminal window
         (this fills in the folder path for you)
      4. Press Enter
      5. Type: ./start.sh
      6. Press Enter
      7. Your browser will open automatically to the tool


   ╔═══════════════════════════════════════════════════════════════════════
   ║                                                                       
   ║   ⚠  DO NOT drag index.html into your browser.                        
   ║                                                                       
   ║   It won't work that way. You MUST use start.bat or start.sh          
   ║                                                                       
   ╚═══════════════════════════════════════════════════════════════════════

How to know it worked:
   - A browser tab opens showing LIFEBOAT with a seal/logo
   - The URL bar says "localhost:3000"
   - The command window is still open (don't close it)

STEP 5: SCAN YOUR WALLET
═══════════════════════════════════════════════════════════════════════════════

Now you're in the tool. You'll see input fields at the top.

   1. COMPROMISED WALLET: Paste the address of your hacked wallet
      (This is the one with NFTs stuck in it)
   
   2. CLEAN WALLET: Paste the address of your safe destination wallet
      (This is where your NFTs will go)
   
   3. ALCHEMY API KEY: Paste the key you copied in Step 3
   
   4. Click "Scan All Chains"
   
   5. Wait. It might take 30 seconds to a minute depending on how many 
      NFTs you have.

How to know it worked:
   - You see your NFTs appear on screen
   - The counter at the top shows how many were found
   - Each NFT has a checkbox next to it


═══════════════════════════════════════════════════════════════════════════════
STEP 6: SELECT WHAT TO RESCUE
═══════════════════════════════════════════════════════════════════════════════

All your NFTs start selected by default. 

   - To REMOVE something from the rescue: Click it to uncheck it
   - To SELECT ALL in a collection: Click the collection header checkbox
   - To FLAG something suspicious: Click the ⚠ button (spam/scam NFTs)

Take your time here. Only rescue what you actually want.

When you're ready, click "Launch Lifeboat" at the bottom.

STEP 8: CHECK BALANCES
═══════════════════════════════════════════════════════════════════════════════

Before executing, click "Check Balances" to verify:

   - The FROM wallet shows your compromised address (probably near zero)
   - The TO wallet shows your funding wallet with enough ETH for gas
   - The gas estimate tells you how much you need

If the funding wallet doesn't have enough ETH, send more before proceeding.

STEP 9: EXECUTE THE RESCUE
═══════════════════════════════════════════════════════════════════════════════

Select the chain you want to rescue first (Ethereum, Polygon, etc.)

Click "Execute Rescue"

The tool will:
   1. Send gas funds privately to your compromised wallet
   2. Fire all NFT transfers privately (hidden from the bot)
   3. Confirm each transfer on-chain

This takes a few minutes. Don't close anything. Watch the progress log.

How to know it worked:
   - You see "🚢 Your lifeboats have reached safety"
   - The confirmed count matches what you selected

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: Browser doesn't open
   → Go to http://localhost:3000 manually

PROBLEM: "Module not found" error
   → Open a command prompt in this folder and run: npm install

PROBLEM: Only Ethereum NFTs show up
   → Enable other chains in Alchemy (see Step 3)

PROBLEM: "Insufficient funds"
   → Your funding wallet needs more ETH/POL/etc for gas

PROBLEM: Transaction failed
   → Some NFTs have transfer restrictions
   → Check the explorer link in the log to see what happened
   → Your rescue data is saved — you can try again

Still stuck? Click the FAQ tab inside the tool.
